# F1-website

A Formula One Website for school to see driver/constructor-standings, race-scheduele, registrer/login and hopefully in the future see more f1 news and other members. 

## How to Start & Use:

npm install

npm install -g json-server

npm run dev

npm run backend

### Built with:

- React
- React Router Dom
- Json-Server
- Json-Server-Auth
- Framer Motion
- React Icons
